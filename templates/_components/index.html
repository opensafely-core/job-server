    • children: [html] - display the children components

    Modal

    • button_text: [string] - the text to display on the button to open the dialog
    • button_tooltip: [string] - the tooltip to display on the button to open the dialog. If not passed there is no tooltip.
    • button_variant: [string] - the button variant (default: primary)
    • button_small: [boolean] - if True, will reduce the font size, weight, and padding
    • button_class: [string] - extra classes to add to the button (default: flex-shrink-0)
    • children: [html] - display the children components. You could include one button with type="cancel", which will close the modal window.
    • custom_button: [string] - add a fragment for a custom button. This button should have the attribute data-modal=id` in order to trigger opening the dialog modal.
    • id: [string] - HTML element ID for dialog element
    • modal_class: [string] - add HTML classes to the dialog HTML component
    • modal_container_class: [string] - add HTML classes to the container wrapping the children and modal
    • request: [Request] - pass the current request into the component. It does not have access by default, and needs access to request.csp_nonce for inline scripting.
    • autoshow: [bool] - open the modal immeadiately. Useful for HTMX use cases.

    Tooltip

    Tooltips should be used sparingly. They are difficult for users to discover and can only show a small amount of information.

    Where possible you should only enhance the interface with a tooltip. An example would be providing the exact time something happened, when hovering over a relative timestamp.

    You should also consider adding an obvious clickable element to attach the tooltip to, enabling users to see there is an action that can be taken to reveal the tooltip.

    • class: [string] - add HTML classes to the component
    • content: [string] - text for the tooltip to display when hovered
    • tooltip_top: [boolean] - if true, show the tooltip on top
    Hover here to see a tooltip {% tooltip content="I am the tooltip!" %}
